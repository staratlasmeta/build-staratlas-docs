# SAGE

## Introduction

This is an NPM module for interfacing with the SAGE program

`@staratlas/sage` account types:

* `Sector`
* `Star`
* `Planet`
* `Fleet`
* `Ship`
* `FleetShips`
* `StarbasePlayer`
* More...
