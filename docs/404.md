# Redirecting...

If you landed on an old markdown or HTML URL, this page will try to send you to the clean VitePress route.

If nothing happens, head back to the [homepage](/).

<script>
  (() => {
    const { pathname, search, hash } = window.location
    const cleaned = pathname
      .replace(/\/index\.html$/, '/')
      .replace(/\.html$/, '')
      .replace(/\.md$/, '')
      .replace(/\/$/, '') || '/'

    if (cleaned !== pathname) {
      window.location.replace(`${cleaned}${search}${hash}`)
    }
  })()
</script>
